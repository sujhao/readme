## tar命令

- *.tar 用 tar -xvf 解压
- *.gz 用 gzip -d或者gunzip 解压
- *.tar.gz和*.tgz 用 tar -xzf 解压   tar.xz  用 tar -Jxvf 解压
- *.tar.gz和*.tgz 用 tar -xzf 解压   tar.xz  用 tar -Jxvf 解压
- *.tar.bz2用tar -xjf 解压
- *.Z 用 uncompress 解压
- *.tar.Z 用tar -xZf 解压
- *.rar 用 unrar e解压
- *.zip 用 unzip 解压
